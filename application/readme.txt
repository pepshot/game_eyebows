Открывая приложение вы попадаете на стартовое окно. На нем написано про игру.

Сюжет игры: отец сбрил брови своему сыну Сереже, ему это конечно же не нравится.
Поэтому он отправляется в увлекательное путешествие. Он собирает очки для того, чтобы заполучить идеальные брови.
Для этого есть у него есть 3 жизни. Ему необходимо пройти 3 уровня и собрать как можно больше очков.
На одном из уровней он может собрать дополнительную жизнь. Также стоит учитывать то, что в игре есть секретные функции.

Сначала нас преветствует стартовое окно.
Для упрвления ипользуйте стрелочки вверх(^), вниз(v), вправо(<-), влево(->).
Во время игры на фоне будет играть музыка Валентина Стрыкало "Сережа".

Чтобы получить иделаьные брови нужно собрать все очки(пластыри), в остальных случаях можно заполучить только монобровь.

При столкнвении главного героя с врагами, он теряет жизнь.
Если 3-е жизней заканчиваются, то герой умирает и игра заканчивается. На экран выводится банер, которыай повествует об окончании игры.
Также на одном из уровней можно будет собрать дополнительную жизнь.

В игре присутствуют секретная фунция - секретные стены, сквозь которые можно ходить.

Если ты прошел игру, выводится финальный экран.
По окончанию игры результат записывается в файл "result.txt". В него записывает успех прохождения игры, сколько пройдено уровней,
сколько у тебя осталось жизней и сколько собрано очков.
