Открывая приложение вы попадаете на стартовое окно. На нем написано про игру.

Сюжет игры: отец сбрил брови своему сыну Сереже, ему это конечно же не нравится. Поэтому он отправляется в увлекательное путешествие. Он собирает очки для того, чтобы заполучить идеаоьные брови. Для этого есть у него есть 3 жизни. Ему необходимо пройти 3 уровня и собрать как можно больше очков. На одном из уровней он может собрать дополнительную жизнь. Также стоит учитывать то, что в игре есть секретные функции.

Для упрвления ипользуйте стрелочки вверх(^), вниз(v), вправо(<-), влево(->).

Чтобы получить иделаьные брови нужно собрать все очки, в остальных случаях можно заполучить только монобровь.

Если 3-е жизней заканчиваются, то герой умирает и игра заканчивается. На экран выводится банер, которыай повествует об окончании игры.

Если ты прошел игру, выводится финальный экран.
По окончанию игры результат записывается в файл "result.txt".
